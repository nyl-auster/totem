# INSTALL

First install totem-server :
https://github.com/nyl-auster/totem-server
it will run locally on localhost:4000

Then install totem :

```
npm install
```
Dev with local totem-server:
```
npm run dev
```
Dev with prod totem-server
npm start

app is running on : http://localhost:3000
